⏳ 待确认交易

基金的盘中估算 ≠ 真实成交价，待确认交易在净值发布后会自动按真实净值回填。在确认前你可以修改申报金额或撤销。

🚦 基金限购管理

支付宝/天天基金没有公开 API 查限购，需要你看到限额变化时手动维护。系统会在定投时自动按限额扣减本期金额（不影响计划），并在你修改限额时弹窗提示对哪些定投有影响。

📅 定投计划

定投 = 持续扣款计划。设定后，每到扣款日系统会自动登记一笔"待净值确认"的买入，等净值发布后自动回填份额与成本。
